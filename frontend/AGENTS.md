# Frontend Agent Scope (Jules)

## Responsibilities
- Implement UI screens and widgets using Material 3 Expressive.
- Connect BLoCs to UI components.
- Implement adaptive layouts (Desktop/Mobile).
- Handle form validations.

## Technical Constraints
- Follow `.agent/rules/bloc_standards.md`.
- Target: Flutter Web WASM.
- Use `go_router` for navigation.
- No `dart:html`; use `package:web`.
